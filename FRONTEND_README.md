## 🚀 **Getting Started**

### **1. Install Dependencies**
```bash
cd frontend
npm install
```

### **2. Environment Setup**
Create a `.env` file in the frontend directory:
```
VITE_API_URL=http://localhost:5000/api
```

### **3. Start Development Server**
```bash
npm run dev
```
Frontend runs on `http://localhost:5173`

### **4. Build for Production**
```bash
npm run build
```
